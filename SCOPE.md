
# Regen Economic Reboot Framework — Scope Document

This repository represents a living coordination layer for Regen’s transition from a payroll/gas token to an ecological economy asset base.

## Objectives
- Align R&D, Foundation, and community processes under a transparent execution roadmap.
- Integrate GaiaAI simulation workflows for tokenomics, supply, and governance modeling.
- Maintain documentation clarity for developers, partners, and stakeholders.
